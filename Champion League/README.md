#📦 API CRUD de Jogadores de Futebol
Este projeto consiste no desenvolvimento de uma API REST utilizando TypeScript e Express, com o objetivo de gerenciar um conjunto de jogadores de futebol através das operações básicas de um CRUD (Create, Read, Update, Delete).

A aplicação permite cadastrar novos jogadores, consultar jogadores existentes, atualizar suas informações e removê-los do sistema. Cada jogador possui dados como nome, clube, nacionalidade, posição e estatísticas de desempenho.

O projeto foi estruturado seguindo boas práticas de organização de código em camadas (controllers, services e repositories), utilizando tipagem estática do TypeScript para maior segurança e manutenção do código.

##🚀 Funcionalidades
- Criar um jogador
- Listar todos os jogadores
- Buscar jogador por ID
- Atualizar informações de um jogador
- Remover jogador do sistema
##🛠 Tecnologias utilizadas
- TypeScript
- Node.js
- Express
- TSX / ts-node (execução do TypeScript)
- Arquitetura em camadas
##🎯 Objetivo do projeto
Praticar o desenvolvimento de APIs REST com TypeScript, aplicando conceitos de tipagem, organização de código e estruturação de projetos backend.
